# 우선순위 큐


## 주요 메소드
* **insertItem(k,e)** : 키가 k인 원소 e를 큐에 삽입
* element **removeMin()** : 큐로부터 최소 키를 가진 원소를 삭제하여 반환

## 일반 메소드
* intteger **size()** : 큐의 항목 수를 반환
* boolean **isEmpty()** : 큐가 비어 있는지 여부를 반환

## 접근 메소드
* element **minElement()** : 큐에서 최소 키를 가진 원소를 반환
* element **minKey()** : 큐에서 최소 키를 반환
